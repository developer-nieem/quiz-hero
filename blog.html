What is The Differences between Map, filter, foreach , find

  • Map: if We applay Map method of the array , its applaying a function per element on the array then give a return array
  • Filter: if We applay Filter method of the array , and we can put a condition, if condition are true then give return all of them that is true
  • Find: if We applay Find method of the array , and we can put a condition, if condition are true then give a return value and return just first one
  • foreach: if We applay foreach method of the array , this method no return a new array , its just executes each element
